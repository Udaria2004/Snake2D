# Snake2D

A simple Snake game made using SFML and C++

## Build Instructions
C++ Compiler
SFML Lib
Make

Run the make command to generate the .exe file.

## Controls

Use Arrow Keys to move up/down or left/right
Snake can eat the food to grow in length which adds 1 point
